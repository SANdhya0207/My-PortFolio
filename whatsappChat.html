TalkFlow Insights Project


TalkFlow Insights Homepage

This was a project I started in October 2023. It took almost 20 days to complete the project.

The WhatsApp Chat Analyzer is a data analysis tool designed to provide insights and statistics from WhatsApp group or individual chat exports. The project is implemented using Python and leverages Streamlit for creating an interactive web-based user interface. The analysis includes various metrics such as the most talkative user, influencer user, long-winded user, emoji lover user, early bird user, and night owl user. Additionally, the tool offers statistical information like total messages, total words, media shared, links shared, and more.

Key Features:

User Insights: Identify key user behaviors, including the most talkative, influencer, long-winded, emoji lover, early bird, and night owl users.

Statistical Overview:Get a quick summary with total messages, words, media, links shared, and graphical timelines for better visualization.

Word and Emoji Analysis: Analyze communication patterns through the most common words and emojis used by users, both overall and individually.

Sentiment Analysis: Understand the emotional tone of the chat with sentiment distribution and an average sentiment score.

Interactive Interface: a user-friendly web-based interface powered by Streamlit, allowing seamless file uploads for chat analysis.

Technologies:
  • - Python
  • - Streamlit
  • - Matplotlib and Seaborn
  • - VADER Sentiment Analysis
  • - Pandas
  • - URLExtract
