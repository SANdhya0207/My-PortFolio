Book Heaven Online

This was a project I started back in February 2023. It took almost 70 days to complete the project.

This project is a feature-rich Ecommerce store application built using ReactJS, ExpressJS, NodeJS, and MongoDB.

It offers a user-friendly interface with a homepage displaying a wide range of products, along with a filtering system for sorting by price or category. The navbar includes a dropdown menu listing all available product categories. User registration and login functionalities with password encryption using bcrypt and JSON web tokens ensure secure access.

The application provides separate functionalities for admins and clients. Admins can manage products, categories, and orders, enabling them to create, update, and delete products.

Users can add desired products to their cart and make purchases using the Braintree payment gateway, accepting both card and PayPal payments. The implementation of toast notifications enhances the user experience throughout the application.

Technologies:
  • - Express JS
  • - React JS
  • - Node JS
  • - MongoDB
  • - Bcrypt Library
  • - Braintree Payment
