VoteEase Project

This was a project I started in June 2023. It took almost 40 days to complete the project.

The project is an online voting portal built using Express.js, React.js, MongoDB, and Node.js.

It facilitates two types of elections: normal democratic elections and president elections. In the normal democratic elections, common people can see political parties' information, register/login to vote in their constituency, and receive an encrypted email upon successful voting. Votes are stored encrypted using AES algorithm, ensuring security and privacy.

For president elections, only Shabha (Lok Sabha and Rajya Sabha) members can vote for the president. Their details are fetched from a government API, and they need to provide their ID to validate their eligibility. The candidate's votes are increased based on the state the Shabha member belongs to, with the vote count encrypted using AES before storing in the database.

After elections, users can view the election results, providing transparency in the electoral process. The project effectively combines modern technologies, secure data handling, and government APIs to create a reliable and transparent online voting platform.

Technologies:
  • - Express JS
  • - React JS
  • - Node JS
  • - MongoDB
  • - Bcrypt Library
  • - Government APIs
  • - Created APIs
